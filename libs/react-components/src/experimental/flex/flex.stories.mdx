# Flex Row

Provides a quick way to implement a flex row, while allowing the component to enforce consistency.

The component allows you to pass in the maximum number of children that you want to exist within the row and if a `minValue`
parameter is specified, the `flex-wrap` style param will position the children in a manner to allow this constraint to be upheld.
