# Badge

The Badge component aims to implement the 'Badge' specifications on the Alberta Design System.
The badge was designed to contain a text and/or an icon.

**Contributors:** ACO Team

**Peer Dependencies:**

- Material UI `<mat-icon />` is used for icon content; alternatively 'icon' element
  must have the `goa-badge-icon` class.
