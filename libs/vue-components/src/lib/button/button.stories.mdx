## Overview

A button to use for a Government of Alberta microsite.  Contains a customizable service name and home link for the microsite.

## Examples
```
<template>
  <div id="app">
    <goa-button
      buttonType="'primary || 'secondary' || 'tertiary'"
      buttonSize="'small' || 'normal'"
      title="Hovering over"
      v-on:click="() => function()"
    >Click me</goa-button>
  </div>
</template>

<script>
import { GoAButton } from '@abgov/vue-components';

export default {
  name: "App",
  components: {
    'goa-button': GoAButton
  }
};
</script>
```
