# App header

### The header helps users identify where they are and provides a quick, organized way to reach important sections of a website.

[Figma design component](https://www.figma.com/file/3pb2IK8s2QUqWieH79KdN7/%E2%9D%96-Components-%7C-DDI?node-id=7224%3A263785)

#### Custom Content

Custom content can be added to the header by passing it in as a child component(s).

<Story name="Custom Content">
  <GoAMicrositeHeader type="live" />
  <GoAAppHeader url="https://example.com" heading="Ticket and Fine Payments">
    <a href="#">Support</a>
    <GoAAppHeaderMenu heading="Tickets" leadingIcon="ticket">
      <a href="#">Cases</a>
      <a href="#">Payments</a>
      <a href="#">Outstanding</a>
    </GoAAppHeaderMenu>
    <a href="#" className="interactive">
      Sign in
    </a>
  </GoAAppHeader>
</Story>

<Tabs>
  <Tab label="Angular">

            <a href="#">Outstanding</a>
