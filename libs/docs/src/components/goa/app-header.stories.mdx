# App header

### The header helps users identify where they are and provides a quick, organized way to reach important sections of a website.

[Figma design component](https://www.figma.com/file/3pb2IK8s2QUqWieH79KdN7/%E2%9D%96-Components-%7C-DDI?node-id=7224%3A263785)

---

#### PropertiesSet the URL to link from the alberta.ca logo. A full url is required

<Props>
  <Prop
    name="url"
    type="string"
    description="Set the URL to link from the alberta.ca logo. A full url is required"
  />
  <Prop
    name="heading"
    type="string"
    description="Set the service name to display in the app header"
  />
  <Prop
    name="maxContentWidth"
    type="string"
    defaultValue="100%"
    description="Maximum width of the content area"
  />
</Props>
