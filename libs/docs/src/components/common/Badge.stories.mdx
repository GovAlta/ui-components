# Badge

Badges are system driven, non-interactive, small labels
which display minimal amounts of information, system feedback, or states.

[Figma design component](https://www.figma.com/file/qWGWDvULAAZuVaDpXCRzqP/DS-Website?node-id=820%3A11807)

---

#### Properties

<Props>
  <Prop
    name="type"
    type="success | warning | information | emergency | dark | midtone | light"
    required={true}
    description="Defines the context and colour of the badge"
  />
  <Prop
    name="content"
    type="string"
    description="Badge text"
  />
  <Prop
    name="icon"
    type="boolean"
    default={false}
    description="Flag to show the icon corresponding to the defined type"
  />
</Props>
