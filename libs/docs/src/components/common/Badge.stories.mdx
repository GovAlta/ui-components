# Badge

Badges are system driven, non-interactive, small labels
which display minimal amounts of information, system feedback, or states.

[Figma design component](https://www.figma.com/file/qWGWDvULAAZuVaDpXCRzqP/DS-Website?node-id=820%3A11807)

---

#### Properties

<Props>
  <Prop
    name="type"
    type="success | important | information | emergency | dark | midtone | light"
    required={true}
    description="Define the context and colour of the badge"
  />
  <Prop
    name="icon"
    type="boolean"
    defaultValue="false"
    description="Include an icon in the badge"
  />
  <Prop name="content" type="string" description="Text label of the badge" />
</Props>
