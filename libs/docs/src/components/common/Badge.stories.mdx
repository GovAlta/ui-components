# Badge

Badges are system driven, non-interactive, small labels
which display minimal amounts of information, system feedback, or states.

[Figma design component](https://www.figma.com/file/qWGWDvULAAZuVaDpXCRzqP/DS-Website?node-id=820%3A11807)

<GoACallout heading='Badge "role" attribute'>
  This badge component does not include the "role" attribute for better parsing
  by screen readers. As a result, it is not fully accessible in situations with
  dynamic content.
</GoACallout>

<GoADetails heading="What is dynamic content?">

<br />
<GoACallout type="important">
  Include an 'arialabel' attribute whenever you’re using an icon-only badge.
  This 'arialabel' should provide a succinct, meaningful description of the
  badge’s purpose or the information it represents. Any icon-only badge without
  an 'arialabel' will be overlooked by screen readers.
</GoACallout>
