# Callout

### Callouts communicate important changes or facts within the body layout through a strong visual emphasis, so that users take notice and read the information.

[Figma design component](https://www.figma.com/file/3pb2IK8s2QUqWieH79KdN7/Components---DIO-DS?node-id=434%3A14140)

---

#### Properties

<Props>
  <Prop
    name="type"
    type="information | important | emergency | success | event"
    required="true"
    description="Define the context and colour of the callout"
  />
  <Prop name="heading" type="string" description="Callout heading text" />
</Props>

#### Emergency

Emergency callout are used to alert the user to only the most important pieces of information.
They can also be used to alert the user to system errors.
