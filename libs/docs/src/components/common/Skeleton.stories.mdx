# Skeleton

### A skeleton provides visual feedback to users while a content heavy page or element is loading.

[Figma design component](https://www.figma.com/file/ylmHeuDMfxnDBnP1VaQYz8/%E2%9D%96-Guidelines-%7C-DIO-DS?node-id=622%3A13702)

---

#### Properties

<Props>
  <Prop
    name="type"
    type="image | text | title | text-small | avatar | header | paragraph | thumbnail | card | profile"
    required="true"
    description="Pre-set skeleton shapes to represent your content"
  />
  <Prop
    name="size"
    type="1 | 2 | 3 | 4"
    defaultValue="1"
    description="Size can affect either the height, width or both for different skeleton types."
  />
  <Prop
    name="lineCount"
    type="number"
    defaultValue="3"
    description="Used within components that contain multiple lines. Currently only used in `card` skeleton type"
  />
  <Prop
    name="maxWidth"
    type="string"
    defaultValue="320px"
    description="Set component maximum width. Currently only used in `card` skeleton type"
  />
</Props>
