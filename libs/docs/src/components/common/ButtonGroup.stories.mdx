# Button Group

Button group displays multiple related actions stacked or in a horizontal row to help with arrangement and spacing.

[Figma design component](https://www.figma.com/file/3pb2IK8s2QUqWieH79KdN7/%E2%9D%96-Components-%7C-DDI?node-id=11627%3A391202)
