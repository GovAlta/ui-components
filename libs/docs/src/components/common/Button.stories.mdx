# Button

### Use a button to carry out an important action or to navigate to another page. See also: <a href="https://www.figma.com/file/UbIKwjbxuPUvoFNYvMBGpe/%E2%9D%96-Patterns-%7C-DDI-DS?node-id=1792%3A20682">button placement</a>

[Figma design component](https://www.figma.com/file/3pb2IK8s2QUqWieH79KdN7/%E2%9D%96-Components-%7C-DDI?node-id=5910%3A235039)

#### Types

There are three main button types: Primary, Secondary, and Tertiary. If there is only one button on a page, it should be a primary button. There should only ever be one primary button on a page
