# App Header Menu

Used within the AppHeader to display a group of menu items

```html
<goa-app-header heading="MyApp">
  <a href="#link">Link</a>
  <goa-app-header-menu heading="Account">
    <a href="#settings">Settings</a>
    <a href="#signout">Signout</a>
  </goa-app-header-menu>
</goa-app-header>
```
