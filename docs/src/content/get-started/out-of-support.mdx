<goa-text size="heading-xl" mt="none" mb="m">Out-of-support versions</goa-text>
<goa-text size="body-l" mt="none" mb="2xl">With the launch of the new Design System, support for Angular v4 and lower, and React v6 and lower is significantly reduced.</goa-text>

<goa-text as="h2" id="what-this-means" size="heading-m">What this means</goa-text>

<goa-text size="heading-xs">Fewer fixes or patches</goa-text>
<p>Bug fixes and security updates are no longer prioritized for these versions.</p>

<goa-text size="heading-xs">No new features</goa-text>
<p>All new features and enhancements are released only in the current major version.</p>

<goa-text size="heading-xs">Use at your own risk</goa-text>
<p>Existing projects may continue to run. However:</p>
<ul>
  <li>Compatibility with new browsers or operating systems is not guaranteed</li>
  <li>Compatibility with future framework updates is not guaranteed</li>
  <li>Issues discovered in these versions will not be resolved</li>
</ul>

<goa-callout version="2" type="emergency" emphasis="low" heading="Active maintenance has ended" mb="m">
  <goa-text size="body-m" mt="2xs" mb="none">Projects using Angular v4 and lower, and React v6 and lower will continue to function. New issues or defects identified in these versions may not be fixed.</goa-text>
</goa-callout>
<goa-text as="h2" id="what-to-do-next" size="heading-m">What to do next</goa-text>
<p>
  Upgrade to the latest supported version. Refer to the
  <a href={withBase(`/get-started/developers/setup`)}>Developer setup page</a>.
</p>
<p>
  If you need support planning your upgrade,
  <a href="https://outlook.office365.com/book/BKGDesignsystemdropinhours@abgov.onmicrosoft.com/" target="_blank">book time in drop-in hours</a>.
</p>
