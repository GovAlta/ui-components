3815 - Date picker size prop

GoabDatePicker now exposes a size prop ("default" | "compact") that applies to both the calendar and input variants.

Known follow-up: when type="input" is used in a narrow container, the three fields still overflow instead of staying on one line (see the last example). The responsive fix is not part of this change.
