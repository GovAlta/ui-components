Feature 3873 — GoabThemeService demo This page demonstrates the Angular theme service shipped from @abgov/angular-components. No manual document.documentElement.setAttribute in this component — the service owns all DOM and localStorage side effects. The service exposes a signal. Current value: {{ theme.mode() }}. Re-render is tracked automatically by Angular when the signal changes. Controls Toggle ({{ theme.mode() === 'light' ? 'to dark' : 'to light' }}) Force light Force dark How to consume Inject the service, bind to the signal in the template: 
theme = inject(GoabThemeService);
// template: {{ theme.mode() }}
// call:     theme.toggle() / theme.setMode('dark')
What the service does On first inject, reads initial mode from: localStorage → existing data-theme attribute → system preference (prefers-color-scheme) → default light. On every change, an effect() syncs data-theme on <html> and persists to localStorage. Refresh the page — mode persists.
